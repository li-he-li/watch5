Deletes any[Record](https://developer.android.com/reference/kotlin/androidx/health/connect/client/records/Record)of type[T](https://developer.android.com/reference/kotlin/androidx/health/connect/client/package-summary#(androidx.health.connect.client.HealthConnectClient).deleteRecords(androidx.health.connect.client.time.TimeRangeFilter))in the given[timeRangeFilter](https://developer.android.com/reference/kotlin/androidx/health/connect/client/package-summary#(androidx.health.connect.client.HealthConnectClient).deleteRecords(androidx.health.connect.client.time.TimeRangeFilter))(automatically filtered to[Record](https://developer.android.com/reference/kotlin/androidx/health/connect/client/records/Record)belonging to the calling application). Deletion of multiple[Record](https://developer.android.com/reference/kotlin/androidx/health/connect/client/records/Record)is executed in a transaction - if one fails, none is deleted.  

```kotlin
import androidx.health.connect.client.deleteRecords
import androidx.health.connect.client.records.StepsRecord
import androidx.health.connect.client.time.TimeRangeFilter

healthConnectClient.deleteRecords<StepsRecord>(
    timeRangeFilter = TimeRangeFilter.between(startTime, endTime)
)
```

)
